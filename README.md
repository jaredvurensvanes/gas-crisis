# GasCrisis.net - Anaesthesia Crisis Manual

A comprehensive web-based reference tool for anesthesia and critical care physicians managing operating theater emergencies. Provides rapid-access clinical decision-making guides organized by the ABCDE framework (Airway/Breathing, Circulation, Disability, Extras, Obstetrics/Pediatrics).

## Features

- **45+ Crisis Scenarios**: Detailed management protocols for common anesthesia emergencies
- **ABCDE Organization**: Structured by physiological systems for quick navigation
- **Search Functionality**: Find crises by keyword
- **Pediatric Parameters**: Weight-based dosing tables
- **Responsive Design**: Works on desktop and mobile devices
- **Offline Capable**: Static site with no external dependencies

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/gascrisis.git
   cd gascrisis
   ```

2. Open `index.html` in a web browser (works with file:// protocol for local development)

3. For production deployment, serve the static files via any web server (Apache, Nginx, GitHub Pages, etc.)
